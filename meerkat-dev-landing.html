Never blocks. Always guides.

Meerkat catches errors without disrupting your agent's workflow. Every interception includes remediation so your agent self-corrects in milliseconds.

INGRESS: SANITIZE + CONTINUE

Malicious injection in an email? Meerkat removes the injection, preserves the safe content, and returns a sanitized version. Your agent processes the clean email without interruption. The threat is logged to the audit trail.

EGRESS: CORRECT + RETRY

Hallucinated a medication dose? Meerkat returns the exact error, what the source actually says, and an instruction your agent can use to self-correct. The agent retries, passes, and executes. User never sees the error.

# Shield: scan incoming content for prompt injection
curl -X POST https://api.meerkatplatform.com/v1/shield \
  -H "Authorization: Bearer $MEERKAT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"input": "email body or web content here"}'

# Verify: check AI output before executing
curl -X POST https://api.meerkatplatform.com/v1/verify \
  -H "Authorization: Bearer $MEERKAT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "input": "user instruction",
    "output": "what the AI is about to do",
    "context": "source data to verify against",
    "domain": "healthcare"
  }'
